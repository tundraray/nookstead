# Механика 08: Квесты, события и сюжетные акты

## Цель

Дать игроку направленный прогресс, который сочетает ручной нарратив и эмерджентные задачи из состояния мира.

## Игровой цикл

1. Игрок получает задачу (сюжет, доска, NPC-органика).
2. Выполняет действия в связке с другими механиками.
3. Получает награду и изменение состояния мира.
4. Открываются новые задачи, регионы и сюжетные слои.

## Основные правила

- Есть 3 класса задач:
- `story` (актовый контент),
- `daily` (регулярная активность),
- `organic` (рождаются из NPC-памяти/событий).
- У задачи есть условия старта, дедлайн, награда, последствия провала.
- Сюжетные ветки могут изменять серверный контекст.

## Состояния и данные

- `QuestDefinition`: type, requirements, steps, rewards.
- `QuestProgress`: playerId, questId, currentStep, status.
- `WorldEvent`: trigger, window, participants, outcome.

## Параметры баланса

- Длина daily: 5-20 минут.
- Длина story-chain узла: 15-45 минут.
- Доля organic-квестов: 10%-30% от всех активных задач.
- Награды: валюта + репутация + unlock-триггеры.

## MVP-срез

- Онбординг-цепочка на 30-45 минут.
- Ежедневная доска из 3 задач.
- 1 тип органического квеста от NPC.

## Телеметрия

- `quest_started`, `quest_step_completed`, `quest_completed`, `quest_failed`.
- `event_joined`, `event_completed`.
- `story_gate_reached`.

## Риски и анти-абьюз

- Перегрузка квестлогом: лимит одновременно активных задач.
- Фарм daily без вариативности: ротации и контекстные модификаторы.
- Ломка сюжета из-за коопа: серверные проверки последовательности шагов.
