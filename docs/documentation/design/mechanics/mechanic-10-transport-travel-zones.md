# Механика 10: Транспорт, путешествия и зоны

## Цель

Организовать перемещение по миру через управляемые маршруты без глобального pathfinding по всей карте.

## Игровой цикл

1. Игрок/NPC выбирает цель в текущем или другом регионе.
2. Для межрегионального маршрута выбирается `TravelNode`.
3. Сущность движется к узлу.
4. Переходит в `IN_TRANSIT`.
5. По `arrivalTime` появляется в целевом узле.

## Основные правила

- Мир разделён на регионы, регион включает чанки.
- Внутри региона доступно обычное локальное перемещение.
- Между регионами перемещение только через `TravelNode`.
- Нет глобального A* через все регионы.

## Состояния и данные

- `TravelNode`: `id`, `regionId`, `chunkId`, `position`, `connectedTo[]`, `travelTime`, `travelCost`.
- `TravelRequest`: `fromRegion`, `toRegion`, `sourceNode`, `targetNode`.
- `travelState`: `NORMAL`, `MOVING_TO_NODE`, `IN_TRANSIT`.
- `TransitRecord`: `entityId`, `targetRegionId`, `arrivalTime`.

## Параметры баланса

- Время транзита: 1-120 минут (по типу маршрута).
- Стоимость: 0 до medium/high в зависимости от дистанции и типа.
- Частота миграции NPC между регионами ограничена системными правилами.

## MVP-срез

- Внутрирегиональное перемещение.
- 1-2 межрегиональных маршрута через TravelNode.
- TransitManager с выдачей в целевом узле по времени.

## Телеметрия

- `travel_node_entered`.
- `transit_started`, `transit_completed`, `transit_failed`.
- `region_changed`.

## Риски и анти-абьюз

- Дюп сущностей в транзите: единый источник истины в `TransitManager`.
- Потеря состояния при перезапуске: персист транзита + восстановление на boot.
- Эксплойт бесплатных маршрутов: лимиты/стоимость/условия доступа.
