# Спецификация UI/UX для MVP Nookstead (Phase 1)

## Обзор

Данный документ определяет полную спецификацию пользовательского интерфейса и опыта для MVP Nookstead -- 2D пиксельной уютной life sim с AI NPC. Документ охватывает все экраны, элементы, анимации, цвета, размеры и взаимодействия, необходимые для Phase 1 (Vertical Slice).

## Часть 1: Принципы дизайна и визуальный язык

### 1.1 Философия "Уютного UI"

Интерфейс Nookstead -- это не "игровой HUD". Это **записки на пергаменте, прикреплённые к кухонной пробковой доске фермерского дома**. Каждая панель, каждая кнопка должна ощущаться как часть уютного мира, а не как наложенная технологическая рамка.

**Шесть принципов:**

| # | Принцип | Реализация |
|---|---------|------------|
| 1 | **Мир доминирует** | HUD занимает не более 15% площади экрана. Полупрозрачные панели. Авто-скрытие при бездействии (fade до 50% через 10с). |
| 2 | **Тепло, не холод** | Никаких металлических или неоновых цветов. Только тёплые земляные тона, мягкие пастели, пергаментные текстуры. |
| 3 | **Мягкие формы** | Скруглённые углы (через 9-slice LimeZu фреймов). Никаких острых линий толще 1px. Тени -- мягкие, 50% непрозрачности. |
| 4 | **Нежные движения** | Все переходы -- плавные (200-400мс). Bounce вместо snap. Fade вместо cut. Никаких резких появлений. |
| 5 | **Звуковая обратная связь** | Мягкий клик (дерево по дереву) при нажатии. Тихий свист при открытии меню. Звон монетки при заработке. |
| 6 | **Уважение к вниманию** | Максимум 3 уведомления одновременно. Информация "по запросу" (hover/tap), а не навязана. |

### 1.3 Типографика

#### Шрифты

| Шрифт | Назначение | Загрузка |
|-------|------------|----------|
| **Press Start 2P** | Заголовки панелей, значения валюты, номера слотов, имена NPC | `next/font/google`, variable `--font-pixel` |
| **Inter** (или system sans-serif) | Текст диалогов NPC, описания предметов, тултипы, длинный текст | `next/font/google`, variable `--font-body` |

**Обоснование:** Пиксельный шрифт для всех длинных текстов нечитаем. NPC-диалоги -- центральный опыт игры. Чистый sans-serif для тела текста + пиксельный для заголовков/чисел = идеальный баланс уюта и читаемости.

### 1.4 Стандарт анимаций

Все анимации в UI Nookstead следуют единому стандарту "нежного движения":

```
┌──────────────────┬──────────┬──────────────────┬─────────────────┐
│ Категория        │ Длитель. │ Easing           │ Пример          │
├──────────────────┼──────────┼──────────────────┼─────────────────┤
│ Мгновенная       │ 50-100мс │ ease-out         │ Нажатие кнопки  │
│ Быстрая          │ 150-200мс│ ease-out         │ Выбор слота     │
│ Стандартная       │ 250-300мс│ ease-in-out      │ Открытие панели │
│ Медленная        │ 400-600мс│ ease-in-out      │ Fade диалога    │
│ Атмосферная      │ 1000-2000│ ease-in-out      │ Смена дня/ночи  │
│ Зацикленная      │ 2000-8000│ linear           │ Пульс энергии   │
└──────────────────┴──────────┴──────────────────┴─────────────────┘
```

**Правило:** Ни одна анимация интерфейса не длится больше 600мс (кроме атмосферных и зацикленных). Пользователь не должен "ждать" UI.

### 2.3 Элемент: Валюта (top-right)

```
+-------------------+
| [Coin] 1,250      |
| [Star]    15      |
+-------------------+
  46x28 @ (430, 4)
```

| Параметр | Значение |
|----------|----------|
| Позиция | top-right, (430, 4) |
| Размер | 46x28 (авто по контенту) |
| Фон | 9-slice SLOT_NORMAL |
| Иконка монеты | 16x16 золотая монета |
| Иконка звезды | 16x16 синяя звезда |
| Текст суммы | Press Start 2P, `#DAA520` (золотой), 5px h, выровнен вправо |
| Разделитель тысяч | Запятая ("1,250") |

**Анимации заработка (+монеты):**
1. Текст суммы окрашивается в `#5FAD46` (зелёный) на 800мс
2. Число подпрыгивает вверх на 4px (150мс ease-out), возвращается (250мс ease-in-out)
3. Плавающий текст "+{сумма}" появляется на 8px выше, зелёный, дрейфует вверх 16px за 1200мс, fade из 100% в 0%
4. При сумме >= 50: иконка монеты вращается 360 градусов (400мс)
5. При сумме >= 100: 3-5 частиц монеток (8x8) летят по дуге Безье к счётчику (600мс)

**Анимации траты (-монеты):**
1. Текст окрашивается в `#C0392B` (красный) на 800мс
2. Число трясётся горизонтально +/-2px (300мс, 3 осцилляции)
3. Плавающий текст "-{сумма}" дрейфует вниз 12px за 1000мс, fade

**Недостаточно средств:**
1. Текст мигает красным 3 раза (600мс)
2. Число трясётся +/-3px (400мс, 3 осцилляции)
3. Тултип "Недостаточно монет!" (красный, 2с auto-dismiss)
4. Звук: мягкий гудок ошибки

**Когда показывать:** Всегда. При диалоге -- fade до 30%. При входе в магазин -- пульс золотого свечения на рамке (2с цикл).

```

**Взаимодействия:**
- Клавиши 1-9, 0: выбор слота (bounce 100мс: scale 1.0 -> 1.15 -> 1.083)

- Click: выбор слота
- Drag & drop: зажать 200мс (desktop) / 400мс (mobile) -> перетаскивание
- Активация предмета: click в мире с выбранным инструментом -> bounce иконки (150мс: 1.0 -> 0.85 -> 1.0)
- Cooldown: radial wipe 200-500мс (соответствует длительности анимации инструмента)

**Прочность инструмента:**
```
Горизонтальная полоска под иконкой, 40x4px:

  59-30%:  #FFC107 (жёлтый)
  29-10%:  #FF9800 (оранжевый)

**Когда показывать:** Всегда, кроме кат-сцен. При диалоге -- fade до 30%. При рыбалке -- slide вниз за экран (200мс).

**Typewriter-эффект:**

### 3.11 Анимация открытия/закрытия диалога

**Открытие (500мс total):**
1. `0мс`: HUD fade до 30% opacity (300мс ease-out)
2. `0мс`: Тёмный overlay fade-in (300мс, до 10% opacity)
3. `100мс`: Окно диалога slide-up от нижнего края (400мс, ease-out-cubic)
4. `300мс`: Портрет NPC fade-in + slight bounce (200мс)
5. `400мс`: Текст NPC начинает typewriter
6. `500мс`: Quick reply кнопки fade-in (200мс)

**Закрытие (400мс total):**
1. `0мс`: Quick replies fade-out (100мс)
2. `50мс`: Окно slide-down (300мс, ease-in-cubic)
3. `100мс`: Overlay fade-out (300мс)
4. `100мс`: HUD fade обратно до 100% (300мс ease-in)

| Параметр | Значение |

| Спрайт | Финальный (зрелый) спрайт выбранной культуры |

| Позиция | На тайле под курсором |
| Условие | Тайл вспахан + пустой + семена в руках |
| Невалидный тайл | Ghost спрайт красный (`#C0392B` tint), 30% opacity |

### 5.3 Анимация полива

```
Последовательность (350мс total):
1.  0мс: Персонаж играет анимацию полива (4 фрейма, 350мс)
2.  100мс: Иконка капли воды (💧 8x8, голубая #5DADE2) появляется
         над грядкой, bounce вниз
3.  200мс: Тайл грядки меняет цвет (сухой -> мокрый, 200мс transition)
4.  350мс: Иконка капли fade-out (300мс)
5.  500мс: Спрайт культуры возвращает полную насыщенность
```

### 5.4 Анимация сбора урожая

```
Последовательность (400мс total):
1.  0мс: Персонаж играет анимацию сбора (3 фрейма, 250мс)
2.  100мс: Спрайт культуры подпрыгивает вверх 8px (100мс ease-out)
3.  200мс: Спрайт scale 1.0 -> 0.0 (200мс ease-in) -- "втягивается"
4.  200мс: Частицы (3-5 leaf particles, зелёные, 4x4) разлетаются
5.  300мс: Иконка предмета (+1 Томат) fly-to hotbar (300мс дуга Безье)
6.  400мс: Тайл возвращается к состоянию "вспаханная" (или "земля")
7.  400мс: "+{количество}" плавающий текст (зелёный, fade 1с)
```

## Часть 10: Доступность (Accessibility)

### 10.1 Контрастность цветов (WCAG AA)

| Пара | Соотношение | Стандарт | Статус |
|------|-------------|----------|--------|
| `#3B2819` на `#F2E2C4` (текст на пергаменте) | 8.2:1 | AA (4.5:1) | Проходит |
| `#6B4226` на `#F2E2C4` (рамки на пергаменте) | 4.6:1 | AA (3:1 для графики) | Проходит |
| `#DAA520` на `#F2E2C4` (золото на пергаменте) | 2.8:1 | AA (3:1) | Не проходит -- исправление: использовать `#B8860B` (DarkGoldenrod, 4.1:1) для текста |
| `#FFF8EC` на `#6B4226` (светлый на ореховом) | 5.3:1 | AA (4.5:1) | Проходит |
| `#C0392B` на `#F2E2C4` (красный на пергаменте) | 4.7:1 | AA (4.5:1) | Проходит |
| `#5FAD46` на `#F2E2C4` (зелёный на пергаменте) | 2.5:1 | AA (3:1) | Не проходит -- исправление: использовать `#3D7A2E` (4.5:1) для текста |

**Исправления:**
- Золотой текст на пергаменте: `#B8860B` (DarkGoldenrod) вместо `#DAA520`
- Зелёный текст на пергаменте: `#3D7A2E` (тёмный луг) вместо `#5FAD46`
- Зелёный fill (энергия, прогресс) остаётся `#5FAD46` -- не текст, а графический элемент

### 10.3 Навигация клавиатурой

| Клавиша | Действие | Контекст |
|---------|----------|----------|
| 1-9, 0 | Выбор слота hotbar 1-10 | Всегда (кроме диалога, ввода) |

| I | Инвентарь toggle | Всегда (кроме диалога, рыбалки) |

| Q | Quick-drop 1 единицу активного | При удержании предмета |

**Focus management:**

- При закрытии -- focus возвращается к предыдущему элементу
- Tab-order соответствует визуальному порядку

### 10.5 Режим "Без анимаций" (Reduced Motion)

```
Настройка в меню: "Уменьшить анимации"
  [Вкл] / [Выкл]

Также автоматически активируется при:
  @media (prefers-reduced-motion: reduce)
```

**Что меняется:**
- Все pulse/shake/bounce анимации отключены
- Переходы мгновенные (0мс вместо 200-600мс)
- Частицы (монеты, листья, снег) отключены
- Cooldown: border-progress вместо radial sweep
- Typewriter: мгновенный показ текста (или 10мс/символ)
- Виньетка энергии: статическая (без fade)
- Weather HUD particles: отключены

### 10.6 Цветовая слепота

Все цветовые индикаторы дублируются другим каналом:

| Элемент | Цвет | Дублирующий индикатор |
|---------|------|-----------------------|
| Энергия (норма) | Зелёный | Полная полоска |
| Энергия (средне) | Жёлтый | Полоска ~50% |
| Энергия (низко) | Оранжевый | Пульс + капля пота |
| Энергия (крит.) | Красный | Тряска + виньетка + замедление |
| Заработок | Зелёный | Стрелка вверх + "+" |
| Трата | Красный | Стрелка вниз + "-" |
| Прочность | Цветовая шкала | Числовое значение при hover |

**Будущее (Phase 2+):** Высококонтрастный режим с паттернами вместо цветов для полосок.

### 10.7 Screen reader: особые случаи

- Все декоративные спрайты: `aria-hidden="true"`
- Typewriter-эффект: `aria-live` обновляется только после завершения печати (не посимвольно)
- Анимации монет: `aria-label` обновляется на финальное значение ("Монеты: 1,300")
- NPC-эмоции: `aria-label` включает эмоцию ("Марко (радостный)")
