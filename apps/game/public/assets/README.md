# Game Assets

All sprites are RPG Maker compatible with 48x48 pixel dimensions.

## Directory Structure

- `characters/` — Character sprite sheets (walk cycles, emotes, etc.)
- `tilesets/` — Tileset images for Tiled maps (48x48 per tile)
- `maps/` — Tiled map JSON exports
- `ui/` — UI elements (portraits, icons, frames)
- `audio/` — Sound effects and music
